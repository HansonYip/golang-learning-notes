# 字符

Go语言中，字符型 Rune 与 int32 是等价的，通常表示一个 Unicode 码点。字符值使用两个单引号括起来，如`'A'`，字符值也可以使用一下字符转义：

| 转义字符 | 实际字符 | 备注 |
| :--- | :--- | :--- |
| \a | 响铃 |  |
| \b | 退格 |  |
| \f | 换页 |  |
| \n | 换行 |  |
| \r | 回车 |  |
| \t | 制表符 |  |
| \v | 垂直制表符 |  |
| \\ | 反斜杠 |  |
| \' | 单引号 |  |
| \uAAAA | Unicode 字符 | 对应 16bit 的码点值，A 表示十六进制数字 |
| \UAAAAAAAA | Unicode 字符 | 对应 32bit 的码点值，A 表示十六进制数字 |
